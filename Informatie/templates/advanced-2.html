RGB-LED

Een RGB-LED is een LED die drie kleuren kan weergeven: rood, groen en blauw. Door de intensiteit van deze kleuren te veranderen kan je alle kleuren van de regenboog maken.

Elke kleur in een RGB-led wordt vertegenwoordigd door een specifieke RGB-waarde, uitgedrukt als getallen tussen 0 en 255 voor rood, groen en blauw. Door deze waarden aan te passen, kunnen diverse kleuren en tinten worden gecreÃ«erd. Dit geeft gebruikers de mogelijkheid om nauwkeurig hun gewenste kleur te bereiken, waardoor RGB-leds zeer veelzijdig zijn voor verlichting, visuele effecten en artistieke toepassingen.

Op het breadboard dat verbonden is aan de Arduino, zien jullie een RGB-led. Download dit bestand, open het en upload de code naar de arduino. Vergeet niet om bij 'Select Board' de juiste arduino te selecteren, anders is het niet mogelijk om je code te uploaden. Het ledje zou nu rood licht moeten geven.

Opdracht 2.1

Maak de led groen door de RGB-waarden van dit stuk code aan te passen. Upload je code opnieuw naar de arduino en test of het ledje groen licht geeft. 


void loop() {
  //Pas hier de RGB waarden aan om verschillende kleuren te bekomen
  analogWrite(pin_rood, 255);
  analogWrite(pin_groen, 0);
  analogWrite(pin_blauw, 0);
}
