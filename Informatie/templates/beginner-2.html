Beginner: LED - 2/10

LED

LED staat voor Light Emitting Diode. Dit zijn halfgeleiders die licht uitzenden. De vaste stof fysica hierachter is voor specialisten elektrotechniek. LEDs gedragen zich, eenmaal ze licht geven, als een batterijtje (er staat een constante elektrische spanning over). De stroom erdoor bepaalt de lichtsterkte. Die stroom kunnen we instellen met behulp van een weerstand. Hoe dat moet? Zie het vak “Elektrische netwerken”.

Elk arduino bord heeft altijd een ledje op het bordje dat kan aangestuurd worden met je programma. Dit ledje is verbonden met pin 13. Je kan dit ledje aansturen door de pin 13 op hoog te zetten (HIGH) of op laag (LOW). Als je de pin op hoog zet zal het ledje branden, als je de pin op laag zet zal het ledje uitgaan. Dit ledje kan je terugvinden op het bordje, zoals aangegeven op de foto hiernaast. De knop hieronder zal dit ledje aan of uit zetten. Probeer het maar eens en zoek het lichtje op het bordje!

Code

Voor de geïnteresseerden kan je hieronder al eens de code vinden om het ledje te laten blinken. Door de pin 13 op hoog te zetten zal het ledje branden, door de pin op laag te zetten zal het ledje uitgaan. De delay zorgt ervoor dat de code 1000ms (1 seconde) wacht vooraleer de volgende regel code uitgevoerd wordt. Dit zorgt ervoor dat het ledje 1 seconde aan en 1 seconde uit is. 

void setup() {
  // put your setup code here, to run once:
  pinMode(13, OUTPUT);
}

void loop() {
  // put your main code here, to run repeatedly:
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}
