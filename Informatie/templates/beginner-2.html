Beginner: Led - 2/11

Led

Led staat voor Light Emitting Diode. Leds zijn halfgeleiders die licht uitzenden. De vastestoffysica erachter leer je in het derde jaar elektrotechniek, in het vak "Halfgeleidercomponenten". Leds zijn diodes, wat wil zeggen dat ze de stroom slechts in één richting doorlaten. Hoe meer stroom ze doorlaten, hoe feller het licht. Die stroomsterkte kunnen we instellen met behulp van een weerstand. Hoe dat kan, leer je in het vak “Elektrische netwerken”.

Elk Arduino-bordje heeft een ingebouwd ledje. Dit ledje is intern verbonden met pin 13, zodat je dit ledje kan aansturen door pin 13 op hoog te zetten (HIGH) of op laag (LOW). Als je de pin op hoog zet zal het ledje branden en als je de pin op laag zet zal het ledje doven. Op de foto hiernaast zie je waar je het ledje precies kan vinden. De knop hieronder zal dit ledje aan of uit zetten. Probeer het maar eens en zoek het lichtje op het bordje!

Code

Voor de geïnteresseerden kan je hieronder al eens een ander voorbeeld van code vinden. Deze code laat het ledje blinken. Door de pin 13 op hoog te zetten zal het ledje branden, door de pin op laag te zetten zal het ledje uitgaan. De delay zorgt ervoor dat de code 1000ms (1 seconde) wacht vooraleer de volgende regel code uitgevoerd wordt. Dit zorgt ervoor dat het ledje 1 seconde aan en 1 seconde uit staat. 

void setup() {
  // put your setup code here, to run once:
  pinMode(13, OUTPUT);
}

void loop() {
  // put your main code here, to run repeatedly:
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}
