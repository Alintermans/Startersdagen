Beginner: Led - 2/11

Led

Led staat voor Light Emitting Diode. Leds zijn halfgeleiders die licht uitzenden. De vastestoffysica erachter leer je in het derde jaar elektrotechniek, in het vak "Halfgeleidercomponenten". Leds zijn diodes, wat wil zeggen dat ze de stroom slechts in één richting doorlaten. Hoe meer stroom ze doorlaten, hoe feller het licht. Die stroomsterkte kunnen we instellen met behulp van een weerstand. Hoe dat kan, leer je in het vak “Elektrische netwerken”.

Elk Arduino-bordje heeft een ingebouwd ledje. Dit ledje is intern verbonden met pin 13, zodat je dit ledje kan aansturen door pin 13 op hoog te zetten (HIGH) of op laag (LOW). Als je de pin op hoog zet zal het ledje branden en als je de pin op laag zet zal het ledje doven. Op de foto hiernaast zie je waar je het ledje precies kan vinden. De knop hieronder zal dit ledje aan of uit zetten. Probeer het maar eens en zoek het lichtje op het bordje!

Code

Voor de geïnteresseerden kan je hieronder al een stukje Arduino code vinden. De globale code is omvangrijker, maar dit stukje toont kort enkele specifieke details uit die volledige code. 

void setup() {
 built_in_led = false  // eenmalige initialisatie van de variable built_in_led, deze bepaalt de status van de pin (en van de led!).

// hier staat de code die in een oneindige loop wordt uitgevoerd
void loop{
...
// hiertussen staat er code om na te gaan of er op het scherm op het knopje is gedrukt - indien wel zal de de variable built_in_led van status veranderen (false wordt true, true wordt false).
...
if (built_in_led) {  // hier wordt er nagegaan of de waarde van built_in_led true geworden is
      digitalWrite(LED_BUILTIN, HIGH);   // Arduino code om een pin onder spanning te zetten, waardoor het ledje zal branden
      built_in_led = true;
    } else {
      digitalWrite(LED_BUILTIN, LOW);  // Arduino code om op een pin geen spanning te zetten, waardoor het ledje zal uitgaan
      built_in_led = false;
    }
}
