Opdracht 3: Kleurensensor

Download dit bestand en upload de code naar de arduino. Bevestig kleurensensor op het zwarte vakje.

Opdracht 2.1

Open de 'Serial monitor'. En lees de rood, groen en blauwe waarde af. Deze waarden komen niet overeen met de normale RGB-waarden voor zwart zoals jullie zagen in vorige oefening. Hoe hoger hier de waarde voor rood, groen of blauw is, hoe minder aanwezig die kleur is. Normaal gezien meet je voor zwart 3 waarden op die zeker boven de 2000 zijn. Indien dit niet het geval is, heb je waarschijnlijk de kleurensensor niet goed op het scherm bevestigd.

Opdracht 2.2

Haal lijn 54 tem 57 uit commentaar. Dit betekent dat je '//' voor elke regel weg moet halen. De code moet er nu zo uitzien: 


int match_kleur(int rood, int groen, int blauw) {
  kleur_ID = 0;
  if ((rood > ...) && (groen > ...) && (blauw > ...)) {
      Serial.println(" Zwart");
      kleur_ID = 0;
    }
  // else if ((rood < ...) && (groen >...) && (blauw > ...)) {
  //   Serial.println("Rood");
  //   kleur_ID = 1;

Om de kleur zwart te herkennen moet je op lijn 54 '...' vervangen door de juiste waarde voor rood, groen en blauw.
Tip: zorg voor voldoende marge! Als je voor rood, groen en blauw waarden rond de 4000 krijgt. Geef dan kleinere waarden in.
Voorbeeld:
Als rood>2500 en groen>2500 en blauw>2500, dan hebben we te maken met zwart. Upload je code naar de arduino en test of hij de kleur zwart kan herkennen.

Opdracht 2.3

Maak het vak nu rood door de slider te bewegen. Wanneer je nu op de Serial monitor kijkt mag de arduino niet meer zeggen dat het vakje zwart is. Indien de arduino nog steeds zegt dat het vak zwart is moet je opdracht 2.2 herhalen en de waarden op lijn 54 juist zetten. Normaal gezien is 'groter dan 2500' goed voor rood, groen en zwart. Haal nu lijn 58 tem 61 uit commentaar en probeer zelf goede waarden te vinden voor rood, groen en blauw. Als dit gelukt is zou de arduino nu zowel zwart als rood moeten herkennen.

Opdracht 2.4

Herhaal opdracht 2.3 voor zowel groen (lijn 62 tem 65) als blauw (lijn 66 tem 69).

Opdracht 2.5

Haal de code voor de andere kleuren uit commentaar. Hier hebben wij al waarden ingevuld voor rood, groen en blauw. Deze waarden kunnen toch een beetje verschillen door een verschillend scherm of helderheid. Test of de arduino de 8 verschillende kleuren kan testen en pas indien nodig de waarden aan. Belangrijk: Sla het bestand op, dit heb je later nog nodig!
