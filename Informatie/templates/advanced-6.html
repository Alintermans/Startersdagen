Opdracht 4.4

Pas je code nu aan zodat de servomotor start bij een hoek van 90°. Vervolgens moet hij langzaam draaien naar 120° om daarna terug naar 90° te draaien. Hieronder kan je een beetje inspiratie opdoen. 

              
void loop() {
  if (Serial.available() > 0) {
    char letter = Serial.read();
    if (letter == 's') {
      Serial.println("Servomotor begint nu");
      for (i=90; i<=120; i++) {
        myservo.write(i);
        delay(20);
      }
      delay(200);
      for (i=120; i>=90; i--) {
        myservo.write(i);
        delay(20);
      }
    }
  }
}

Opdracht 4.6

Download dit bestand. Het Chemisch team hebben een opstelling waarmee zij met hun batterij een pico microcontroller kunnnen voeden. Deze pico is verbonden met een OLED scherm en een buzzer. De pico heeft een analoge input die afhankelijk van de waarde van de input een bepaald liedje afspeelt. Er zijn er 8 in totaal. Laad de code op de arduino en probeer de verschillende liedjes af te spelen. Dit kan je doen door 's' te versturen op de serial monitor en vervolgens na de vraag een waarde tussen 0 en 164 te versturen, wat overeenkomt met 0 en 3.3V. Zoek dus de 8 waardes die nodig zijn om de 8 liedjes af te spelen. Deze zal je moeten onthouden voor de volgende opdracht.
