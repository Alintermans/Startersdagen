Opdracht 4.4

Pas de code verder aan zodat de servomotor beweegt aan de helft van de snelheid. 

              
void loop() {
  if (Serial.available() > 0) {
    char letter = Serial.read();
    if (letter == 's') {
      Serial.println("Servomotor begint nu");
      for (i=90; i<=120; i++) {
        myservo.write(i);
        delay(20);
      }
      delay(200);
      for (i=120; i>=90; i--) {
        myservo.write(i);
        delay(20);
      }
    }
  }
}

Opdracht 4.5

Download dit bestand. Team Energie heeft een opstelling waarmee zij met hun batterij een pico microcontroller kunnnen voeden. Deze pico is verbonden met een oled scherm en een buzzer. De pico heeft een analoge input die afhankelijk van de waarde van de input een bepaald liedje afspeelt. Er zijn er 8 in totaal. Laad de code op de arduino en probeer de verschillende liedjes af te spelen. Dit kan je doen door 's' te versturen op de serial monitor en vervolgens na de vraag een waarde tussen 0 en 164 te versturen, wat overeenkomt met 0 en 3.3V. Als richtwaarde kan je respectievelijk 0.5, 0.9, 1.3, 1.7, 2.0, 2.4, 2.8 en 3.3 V nemen voor liedjes 1 tot en met 8. Let op, als je alle waardes gaat willen uitproberen, ga je veel geduld moeten hebben. De pico is zo geprogrammeerd dat het volledige liedje afspeelt (duur +/- 3 minuten per liedje) vooraleer hij zal merken dat er al een volgend liedje (= een ander voltage) gevraagd is. Eventueel kan je de buzzer ontkoppelen (= de grond losmaken van het breadboard) en kijk je wat er verandert op het schermpje.

Deze zal je moeten onthouden voor de volgende opdracht, waarbij je de juiste waardes gaat moeten koppelen aan de juiste kleurencombinatie (= de juiste prof).
